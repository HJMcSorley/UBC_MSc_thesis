this folder ("shimadzu") belongs in the R-inputs directory for Hannah's Master's research ("UBC_MSc/Thesis_UBC-forWater-MSc_HMc/R_UBC-forWater-MSc_HMc/R-inputs_UBC-forWater-MSc_HMc")

This directory contains two sub-directories: "samples" and "results".

samples -- sample data includes order of analysis and sample identifier data in .csv files organized by trip number (field trips to the Leech).
results -- results data was exported from the UBC ESB-3062 EcoHydrology Lab Shimadzu TOC-V autoanalyzer as .txt files (also organized by trip numbers).
